# aviation-airflow

**Data Pipeline using Apache Airflow for Thai Airways Fleet Status Data**

In this project, I need to make a data pipeline using Apache Airflow to get Thai Airways fleet status.

$~$

## Technology Used

- Apache Airflow
- Python Programming
- Docker
- MinIO
- PostgreSQL and MySQL
- MongoDB
